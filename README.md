# 🍉 飞瓜大盘数据采集

Chrome 扩展，自动遍历飞瓜大数据平台类目树，采集销售额、销量、视频/直播数据并实时导出 CSV。

支持 **[飞瓜数据](https://www.bigdatavoice.com)** 和 **[抖衣飞瓜](https://dy.feigua.cn)** 双域名。

## ✨ 功能

- **自动遍历类目** — 从指定起始路径递归遍历所有末级类目，逐级悬停展开，末级点击触发数据加载
- **接口拦截** — 通过 MAIN world 注入拦截 `salesTrendData` 接口响应，精确匹配
- **实时导出 CSV** — 每次点击末级类目后立即写入本地文件，用 Excel 打开即可看到数据逐行增长
- **防封控** — 每次点击后随机等待 3~5 秒
- **多开并行** — 可打开多个标签页，分别选择不同类目和文件名同时采集
- **自定义路径** — 支持指定类目深度，如 `母婴宠物 > 母婴用品` 只遍历该二级分支

## 📊 输出字段

| 字段 | 说明 |
|------|------|
| 类目名称 | 完整类目路径，如 `美妆 > 彩妆香水 > 眼部彩妆 > 睫毛膏` |
| 日期 | 数据日期 `YYYY/MM/DD` |
| 销售额 | 销售额（元，已去除分位） |
| 销量 | 销售件数 |
| 视频销售额 | 视频带货销售额（元） |
| 视频销量 | 视频带货件数 |
| 直播销售额 | 直播带货销售额（元） |
| 直播销量 | 直播带货件数 |
| 浏览量 | 页面浏览量 |

所有金额字段自动截断后两位（分→元），不四舍五入。

## 📦 安装

1. 下载本仓库或 `git clone`
2. 打开 Chrome，地址栏输入 `chrome://extensions`
3. 右上角开启 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择 `feigua_extension` 文件夹
6. 完成，工具栏出现 🍉 图标

## 🚀 使用

1. 打开飞瓜数据概览页：
   - 飞瓜数据：`https://www.bigdatavoice.com/app/#/data-overview/index?tab=goods`
   - 抖衣飞瓜：`https://dy.feigua.cn/app/#/data-overview/index?tab=goods`
2. 确保页面已登录且数据概览页加载完成
3. 点击工具栏 🍉 图标
4. 填写配置：
   - **类目路径** — 每行一个起始路径，`>` 分隔层级
   - **文件名称** — 导出的 CSV 文件名（不含后缀）
5. 点击 **开始遍历**
6. 在弹出的文件夹选择框中选择保存位置
7. 自动开始遍历，面板实时显示进度
8. 用 Excel 打开 `你的文件名.csv` 即可看到数据实时增长

### 类目路径示例

```
个护家清
美妆
母婴宠物 > 母婴用品
母婴宠物 > 宠物食品 > 猫粮
```

- 每行一条起始路径
- `>` 前后空格可选
- 不写层级则遍历该一级类目下的全部末级
- 指定到更深层级则只遍历该分支

### 并行采集

同时打开多个标签页，每个标签页：
1. 选择不同的类目路径
2. 设置不同的文件名（如 `个护家清数据`、`美妆数据`）
3. 分别开始遍历

各标签页数据独立写入各自的 CSV 文件，互不干扰。

## 🏗 架构

```
feigua_extension/
├── manifest.json       # 扩展配置（MV3）
├── popup.html          # 弹窗界面（浅色西瓜主题）
├── popup.js            # 弹窗逻辑，注入脚本
├── interceptor.js      # MAIN world 注入，拦截真实 XHR/fetch
├── content.js          # 核心逻辑：遍历、采集、CSV 写入、面板 UI
├── icon16.png          # 扩展图标 16px
├── icon48.png          # 扩展图标 48px
└── icon128.png         # 扩展图标 128px
```

- **interceptor.js** — 注入页面 MAIN world，hook 真实 `XMLHttpRequest`/`fetch`，匹配 `salesTrendData?periodType=` 接口，响应数据通过 DOM 共享给 content script
- **content.js** — ISOLATED world 运行，控制级联选择器遍历、读取 DOM 中的拦截数据、生成 CSV、通过 File System Access API 实时写入文件

## ⚠️ 注意事项

- 遍历过程中请勿手动操作页面级联选择器，可能干扰自动化流程
- 如遇「未找到父节点」警告，通常是页面渲染延迟，程序会自动跳过并继续
- CSV 使用 UTF-8 BOM 编码，Excel 直接打开可正确显示中文
- 停止遍历时会自动保存已采集的数据
